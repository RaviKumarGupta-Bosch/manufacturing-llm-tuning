# Manufacturing Knowledge Base
## Standard Operating Procedures & Domain Reference

### 1. OEE Reference

**Formula:**
OEE = Availability × Performance × Quality

**World-class benchmarks:**
- OEE > 85% = World Class
- OEE 65–85% = Typical (improvement opportunity)
- OEE < 65% = Significant losses present

**Six Big Losses:**
1. Unplanned Downtime (Availability)
2. Planned Downtime / Changeover (Availability)
3. Minor Stops / Idling (Performance)
4. Reduced Speed (Performance)
5. Startup Rejects (Quality)
6. Steady-State Rejects (Quality)

### 2. Vibration Severity — ISO 10816

| Zone | RMS Velocity | Machine Status |
|------|-------------|----------------|
| A    | < 0.7 mm/s  | New, excellent |
| B    | 0.7–1.8 mm/s | Acceptable for long-term |
| C    | 1.8–4.5 mm/s | Alarm — schedule maintenance |
| D    | > 4.5 mm/s  | Danger — immediate action |

---

### 3. FMEA Risk Priority Number (RPN)

RPN = Severity × Occurrence × Detection (each 1–10)

| RPN Range | Action Required |
|-----------|----------------|
| < 50      | Monitor only |
| 50–100    | Review — consider action |
| 100–200   | Corrective action required |
| > 200     | STOP — must resolve before launch |

Severity 9–10 = Safety/regulatory impact — mandatory independent review regardless of RPN

### 5. Predictive Maintenance Intervals

| Asset Type | Technology | Interval |
|-----------|-----------|---------|
| Motors < 15 kW | Vibration analysis | Monthly |
| Motors 15–75 kW | Vibration + thermography | Bi-weekly |
| Motors > 75 kW | Continuous online monitoring | Real-time |
| Hydraulic systems | Oil analysis | Quarterly |
| Gearboxes | Oil analysis + vibration | Monthly |
| Compressors | Vibration + discharge temp | Weekly |

### 6. Common Defect Categories — Casting

| Defect | Common Cause | Detection Method |
|--------|-------------|-----------------|
| Porosity | Gas entrapment, moisture | X-ray, CT scan |
| Shrinkage | Inadequate feeding | Radiographic inspection |
| Cold shut | Low metal temp, slow fill | Visual, dye penetrant |
| Misrun | Low temp, thin sections | Visual |
| Hot tear | Restricted shrinkage | Visual, dye penetrant |
